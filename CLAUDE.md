**Composants UI :**
- `.card` + `.card-body` → contenus structurés
- `.btn` + variations → boutons
- `.nav` + `.nav-container` → navigation

**Typographie :**
- `.text-body` → paragraphes standard
- `.text-muted` → texte secondaire
- `.fade-in` → animations d'entrée
